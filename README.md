# CRM システム

顧客関係管理 (CRM) システムです。

## 概要

このプロジェクトは、企業の顧客管理、製品管理、店舗管理、店員管理を統合的に行うためのWebアプリケーションです。

## 主要機能

### 認証システム
- ユーザー登録・ログイン・ログアウト
- セッション管理
- パスワードリセット機能
- 認証必須ページ保護

### 顧客管理
- 顧客登録・編集・削除・詳細表示
- 顧客一覧表示とページネーション
- 名前・メール・電話番号での検索機能
- バリデーション（名前・メール必須、メール形式チェック）

### 製品管理
- 製品登録・編集・削除・詳細表示
- 製品一覧表示とページネーション
- 名前・説明・SKUでの検索機能
- 在庫管理機能
- バリデーション（名前・価格必須、価格正数チェック）

### 店舗管理
- 店舗登録・編集・削除・詳細表示
- 店舗一覧表示とページネーション
- 名前・住所・ステータスでの検索機能
- ステータス管理（営業中・閉店・準備中）
- バリデーション（名前・住所必須）

### スタッフ管理
- スタッフ登録・編集・削除・詳細表示
- スタッフ一覧表示とページネーション
- 名前・メール・役職での検索機能
- 店舗との関連付け機能
- ステータス管理（在籍・休職・休暇中）
- バリデーション（名前・メール・役職・入社日必須）

- **マイグレーション**: GORM AutoMigrate

## API仕様

### 認証API
- `POST /api/auth/register` - ユーザー登録
- `POST /api/auth/login` - ログイン
- `POST /api/auth/reset-password` - パスワードリセット要求
- `GET /api/user/profile` - プロフィール取得

### 管理API（認証必須）
- **顧客管理**: `/api/customers/*` - 8エンドポイント
- **製品管理**: `/api/products/*` - 8エンドポイント
- **店舗管理**: `/api/stores/*` - 13エンドポイント
- **スタッフ管理**: `/api/staff/*` - 13エンドポイント

### 主要機能
- 検索・フィルタリング・ページネーション
- CRUD操作（作成・読取・更新・削除）
- バルク操作（一括更新・削除）
- ステータス管理・関連付け機能
