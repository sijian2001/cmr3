# CRM システム

顧客関係管理 (CRM) システムです。

## 概要

このプロジェクトは、企業の顧客管理、製品管理、店舗管理、店員管理を統合的に行うためのWebアプリケーションです。

## 主要機能

### 認証システム
- ユーザー登録・ログイン・ログアウト
- セッション管理
- パスワードリセット機能
- 認証必須ページ保護

### 顧客管理
- 顧客登録・編集・削除・詳細表示
- 顧客一覧表示とページネーション
- 名前・メール・電話番号での検索機能
- バリデーション（名前・メール必須、メール形式チェック）

### 製品管理
- 製品登録・編集・削除・詳細表示
- 製品一覧表示とページネーション
- 名前・説明・SKUでの検索機能
- 在庫管理機能
- バリデーション（名前・価格必須、価格正数チェック）

### 店舗管理
- 店舗登録・編集・削除・詳細表示
- 店舗一覧表示とページネーション
- 名前・住所・ステータスでの検索機能
- ステータス管理（営業中・閉店・準備中）
- バリデーション（名前・住所必須）

### 店員管理
- 店員登録・編集・削除・詳細表示
- 店員一覧表示とページネーション
- 名前・メール・従業員IDでの検索機能
- 店舗との関連付け
- バリデーション（名前・メール・従業員ID必須）
